What do we do?

This website enables you to request a recording session, create a new "ticket" for a multitrack, and view all active tickets. A ticket is a tracking tag on a multitrack as it starts from dry multitrack stems from a recording studio and becomes a multitrack. The lifecycle of a multitrack is as follows:

  1. Request to record a multitrack
  2. Request is approved and recording session is scheduled
  3. Recording session occurs
  4. Track gets mixed
  5. Track gets exported
  6. Track gets run through multitrack app
  7. Track is added to database
