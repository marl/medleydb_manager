Multitrack Processing Instructions

Forms and Junk

  1. If appropriate, ask the folks you are recording if they are willing to let you use their track/s for Medley, and if they agree have them fill out the forms.
  2. Save a copy of their session to the folder MedleyDB/Unprocessed on the server SIXSERVE2/mtechxsan.
  3. Somewhere in that session, save a copy of the metadata form they filled out.

Mixing

  1. Pick a session from the Unprocessed folder, mix it to your heart's content, and then bounce + process it.
  2. Once you've done that, change the color of the session's folder to green.

Bouncing

Bounce all audio to 44.1kHz, 16 bit .wav files.
  1. Make a new folder. Within that folder create a folder called "stems"and a folder called "raw".
  2. Select the mix region (this never changes for the rest of the steps!). All bouncing below should be bouncing the same region.
  3. Bounce the mix to a stereo track and save it to the folder.
  4. Decide how to group the stems into the most logical groupings based on instrument.
  5. Bounce each of the stems to a stereo track with all plugins active. Save everything to the "stems" folder. Name each of them something that you'll remember (eg. volin1.wav, violin2.wav, cello.wav)
  6. Bounce each of the "raw" tracks (only the ones actually used!) to a mono audio file and save them to the raw folder. If it was recorded as stereo, bounce to multiple mono. Name them based on what instrument they get used for (e.g. violin1-1.wav, violin1-2.wav etc). Make sure all plugins are inactive except any delays used for time alignment.
